Simple SOAP Service CXF WSDL2Java project for the Chatroom service.

Requirements:
    Java 7.x or higher
    maven 3.x

Includes:
    - WSDL file.
    - Separated XSD files for the model and the messages.
    - XJB file for the mapping.
    - CXF Maven plugin to generate Java stub/skeleton.

Just run mvn clean install in the main directory.
